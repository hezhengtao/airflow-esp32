## ⚠ 接线前必读 — 极性速查表

下表标 **⚠** 的元件接反会**烧毁器件**，请逐项核对后上电。

| 元件 | 正极/信号 → 接哪里 | 负极/地 → 接哪里 | 接反后果 |
|------|-------------------|------------------|---------|
| **PAM8302 功放** | ⚠ VCC → 5V(+)红线 | GND → GND(-)黑线 | 功放烧毁 |
| | IN+ → RC滤波输出(100Ω后) | IN- → GND | 无声 |
| | SD → 3.3V(使能) | — | 不工作 |
| **470µF/25V 电机** | ⚠ +(长脚) → 电机12V红线 | -(短脚) → 电机GND黑线 | 电容爆裂 |
| **48F704P840 电机** | ⚠ 红线 → 12V(+) | 黑线 → GND(-) | 驱动板烧毁 |
| **Y01 传感器** | ⚠ VCC → 5V(+) | GND → GND(-) | 传感器烧毁 |
| | TX → GPIO42(RX) | RX → GPIO41(TX) | 无数据 |
| **LCD屏幕** | VCC(pin2) → 5V(+) | GND(pin1) → GND(-) | 不亮或损坏 |
| **触摸面板** | VCC → 3.3V | GND → GND(-) | 无响应 |
| **8Ω喇叭** | + → PAM8302 OUT+ | - → PAM8302 OUT- | 相位反(不烧) |

**无极性元件 (正反接都可以):**
- ✅ 100nF 陶瓷电容 (标记"104") ×4 — 两只脚一样
- ✅ 10nF 陶瓷电容 (标记"103") ×1 — 两只脚一样
- ✅ 所有电阻 (1kΩ, 4.7kΩ, 100Ω) — 两只脚一样
- ✅ PT1000 铂电阻 — 两只脚一样

**电解电容口诀：长脚=正极(+) 短脚=负极(-) 外壳有"-"标记一侧是负极**

## 三、系统供电

```
DPC3512 四路电源:
  CH1: 5V/3A  → ESP32-S3 USB-C
  CH2: 12V/2A → 48F704P840 电机 (并联470µF+100nF)
  CH3: 5V/2A  → LCD屏幕 + PAM8302功放 + Y01传感器
  CH4: 3.3V/1A → 仅用于PT1000分压参考 (勿接ESP32 3.3V引脚!)

⚠ 所有模块GND必须汇接在一起(共地)!
⚠ DPC3512内置保护，无需外加保险丝
⚠ ESP32的3.3V由USB 5V经板载LDO提供，CH4仅供PT1000，不可接入ESP32 3.3V引脚以免电源冲突
⚠ 电机12V启动电流大，470µF LOW ESR电解电容提供瞬时补偿
```

## 四、完整接线步骤

> 详细彩色SVG接线图参见同目录 `AirPurifier_Complete_Guide.html`

### 4.1 LCD触摸屏 (TK043F1509，厂家预焊排针)

```
电源:    pin2(VCC) → 5V(+)红线    pin1(GND) → GND(-)黑线
数据:    GPIO4-11 → D0-D7 (8根杜邦线，顺序不能错)
控制:    GPIO12 → CS    GPIO13 → RS    GPIO14 → WR
         GPIO15 → RST   GPIO16 → BL(背光PWM)
触摸I2C: GPIO17 → SDA (经4.7kΩ上拉至3.3V)
         GPIO18 → SCL (经4.7kΩ上拉至3.3V)
         GPIO3  → INT   GPIO21 → RST
         触摸VCC → 3.3V  触摸GND → GND
```

### 4.2 无刷电机 (48F704P840)

```
⚠ 红线=12V(+)  → DPC3512 CH2(+)   ← 反接烧驱动板!
  黑线=GND(-)  → DPC3512 GND(-)
  黄线=START   → GPIO38 (经4.7kΩ电阻下拉至GND，确保默认停机)
  绿线=PWM      → GPIO39 (25kHz速度调节)
  蓝线=FG      → GPIO40 (转速反馈，脉冲/转需实测确认，默认6)

电机12V端子并联:
  ⚠ 470µF/25V电解: +(长脚)焊12V红线, -(短脚)焊GND黑线
  100nF陶瓷: 无极性，同样并联在12V和GND之间
```

### 4.3 Y01空气质量传感器

```
⚠ VCC → 5V(+)红线    GND → GND(-)黑线   ← 反接烧传感器!
  TX  → GPIO42 (ESP32 RX)   交叉连接!
  RX  → GPIO41 (ESP32 TX)   交叉连接!
  VCC和GND之间并联100nF陶瓷退耦电容
```

### 4.4 PT1000温度传感器

```
  3.3V → 1kΩ精密电阻(无极性) → GPIO2(ADC)
  GPIO2 → PT1000(无极性) → GND
  
  分压公式: R_pt1000 = V_adc × 1000 / (3.3 - V_adc) (ADC参考=3.3V时成立)
  ⚠ ESP32-S3 ADC非线性±5%, 如需高精度(±0.5°C)建议用ADS1115等外置ADC
  室温阻值约1070-1100Ω，用万用表验证
```

### 4.5 喇叭音频系统

```
音频链路: GPIO48 → [100Ω电阻串联] → [10nF电容到GND] → PAM8302 IN+

PAM8302功放模块 (Class-D BTL输出, 无需隔直电容):
  ⚠ VCC  → 5V(+)红线           ← 反接烧功放!
     GND  → GND(-)黑线
     SD   → 3.3V (使能，低电平=关断)
     IN+  → RC滤波输出 (100Ω电阻后端)
     IN-  → GND
     OUT+ → 喇叭(+)    ← BTL桥接输出, OUT+/OUT-间无直流偏置, 直接接喇叭
     OUT- → 喇叭(-)

喇叭:
  + → PAM8302 OUT+
  - → PAM8302 OUT-

功能:
  · 按键点击 → "嘀"声 (1kHz, 50ms)
  · 开机     → 上升音阶 ♪ C5→E5→G5
  · 关机     → 下降音阶 ♪ G5→E5→C5
  · 空气质量超标 → 间歇报警 (1kHz/500Hz交替)
  · WiFi配网 → 短促嘀声 (每2秒)
```

### 4.6 杜邦线颜色约定

```
🔴 红色 → 电源正极 (5V, 12V, 3.3V)
⚫ 黑色 → GND 地线 (全部共地)
🔵 蓝色 → LCD 数据/控制信号
🟡 黄色 → 电机控制信号
🟢 绿色 → 传感器信号 + 音频信号
🟣 紫色 → 控制信号 (SD使能, INT等)
```

## 八、故障排查

### 屏幕不亮
```
GPIO16(背光)应为高电平~3.3V | GPIO15(RST)上电后高电平 | CH3输出5V正常 | FPC排线重插
```

### 触摸不响应
```
I2C地址应为0x38 | SDA/SCL不交叉 | 4.7kΩ上拉电阻确认已焊 | ESP32内部上拉不够
```

### 电机不转
```
⚠ 先确认红黑线极性! | 黄线运行时应>2.5V | PWM占空比≥20% | 470µF极性确认
测试: 黄线接3.3V + 绿线接3.3V → 电机应全速转
FG脉冲数: 如转速读数不准, 需用示波器实测FG频率, 调整 MOTOR_PULSES_PER_REV
```

### 喇叭无声
```
检查RC滤波器(100Ω+10nF)是否接好 | PAM8302 SD→3.3V | 
PAM8302 OUT+→喇叭+, OUT-→喇叭- (BTL输出, 已无需隔直电容)
万用表电阻档测喇叭: 应读7-8Ω
```

### 喇叭全是刺耳噪声
```
RC滤波器未接或接错! GPIO48输出的44.1kHz PWM载波未滤除。
必须: GPIO48 → 100Ω电阻 → 10nF电容到GND → PAM8302 IN+
没有RC滤波=喇叭输出的是PWM方波而非音频!
```

### Y01无数据 / PT1000读数异常
```
Y01: TX/RX是否交叉 | 波特率9600 | 预热至少60秒
PT1000: 1kΩ参考电阻值确认(非10kΩ/100Ω) | 分压中点≈1.7V(室温)
⚠ ESP32-S3 ADC线性度: ±5%误差属正常, 如需高精度建议外接ADS1115等16-bit ADC模块
  分压公式 R_pt1000 = V_adc × 1000 / (3.3 - V_adc) 仅在ADC参考电压=3.3V时成立
```
