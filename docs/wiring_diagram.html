分步焊接与接线指南

1

电源准备

  • ESP32-S3 通过 USB-C 供电 (5V)
  • 12V 电源适配器给电机供电
  • 在面包板上建立电源轨: 3.3V / 5V / 12V / GND
  • 焊接 4 颗 100nF (104) 退耦电容, 分别跨接在 ESP32 3.3V ↔ GND, 电机 12V ↔ GND, Y01 5V ↔ GND, LCD 5V ↔ GND
2

LCD 显示屏接线 (8-bit 并口 8080, 共 14 根)

  • LCD 通过 40-Pin FPC 排线 连接到 FPC-to-DIP 转接板
  • D0-D3 → GPIO4 ~ GPIO7
  • D4-D7 → GPIO15 ~ GPIO18
  • CS → GPIO1 (片选)
  • RS → GPIO42 (命令/数据选择)
  • WR → GPIO2 (写使能)
  • RST → GPIO46 (复位)
  • VCC → 5V GND → GND
3

触摸 FT5x06 接线 (I2C, 4 根)

  • 触摸控制器集成在 LCD 模组内, 通过同一 FPC 排线引出
  • SDA → GPIO20
  • SCL → GPIO13
  • 焊接 4.7kΩ 上拉电阻 x2: SDA→3.3V, SCL→3.3V
  • VCC → 3.3V GND → GND
4

BLDC 电机接线 (5 根)

  • START → GPIO38
  • 焊接 4.7kΩ 下拉电阻: GPIO38 → GND
  • PWM → GPIO39 (25kHz 速度控制)
  • FG → GPIO40 (转速反馈脉冲)
  • 电机电源 → 12V, 电机 GND → GND
  • ⚠ 焊接 470uF/25V 电解电容 在电机 12V ↔ GND 之间
5

Y01 空气质量传感器接线 (4 根)

  • TX (GPIO41) → Y01 RX 引脚
  • RX (GPIO9) → Y01 TX 引脚
  • VCC → 5V GND → GND
6

温度传感器接线 (PT1000 / DS18B20 二选一)

  • 方案A PT1000 (模拟): 3.3V → 1kΩ精密 → GPIO3 → PT1000 → GND (分压法, 需标定)
  • 方案B DS18B20 (数字, 推荐): GPIO3 → DS18B20 DQ(黄) | DQ→4.7kΩ→3.3V(上拉) | VDD(红)→3.3V GND(黑)→GND
7

扬声器接线 (3 根 + RC 滤波 + PAM8302)

  • GPIO48 → 100Ω 电阻 → PAM8302 IN+
  • 在 PAM8302 IN+ 处 焊 10nF (103) 电容到 GND
  • PAM8302 VDD → 5V, GND → GND
  • PAM8302 OUT± → 扬声器两端 (BTL, 不接地!)
8

最终检查

  • 所有 GND 必须共地
  • I2C 上拉电阻已焊接 (SDA→3.3V, SCL→3.3V 各 4.7kΩ)
  • 电机 START 下拉已焊接 (GPIO38→GND 4.7kΩ)
  • 退耦电容已就位 (至少 4 颗 100nF)
  • 电解电容极性正确 (470uF 长+短-)
  • 1kΩ PT1000 参考电阻是精密电阻
  • 扬声器 PAM8302 OUT+/OUT- 不接地

⚠ 关键警告:
• 所有 GND 必须共地!
• 电解电容极性不能反! 470uF/25V: 长脚(+)→12V, 短脚(-)→GND。接反会爆炸。
• PAM8302 输出不接地! OUT+ 和 OUT- 直接接扬声器两端。
• Y01 用 5V 供电! 不是 3.3V。
💡 调试建议:
• 先不通电, 用万用表蜂鸣档检查所有 GND 是否互通, VCC-GND 之间没有短路。
• 烧录固件后串口监视器可看启动日志: idf.py -p COM34 monitor。
• 触摸无响应? 检查 I2C 上拉电阻 (4.7kΩ) 是否焊接。
• 电机不转? 检查 GPIO38 下拉电阻, 以及 470uF 电解电容是否正确焊接。
